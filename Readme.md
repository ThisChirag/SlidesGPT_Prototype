# AI Slides Generation Prototype

This is a **prototype** of an AI-powered slides generation application (have used **JavaScript** ). The project demonstrates the basic functionalities of generating slides dynamically, with plans for a fully functional version in the near future.

- This project is currently in its **prototype phase**. A complete working version with more features and optimizations will be released soon. Stay tuned for updates!

### ðŸ”„ Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/ThisChirag/SlidesGpt.git
   cd SlidesGpt
   ```

2. Install backend dependencies:
   ```bash
   cd backend
   pnpm install
   ```

3. Install frontend dependencies:
   ```bash
   cd ../frontend
   pnpm install
   ```

4. Start the backend server:
   ```bash
   cd ../backend
   pnpm start
   ```

5. Start the frontend server:
   ```bash
   cd ../frontend
   pnpm run dev
   ```

6. Open the application in your browser at [http://localhost:3000](http://localhost:3000) (or the specified port).
